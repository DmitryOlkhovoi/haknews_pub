PEP 486 – Make the Python Launcher aware of virtual environments (2015)

PEP 486 предлагает усовершенствование Python Launcher для Windows, чтобы он стал осведомлённым о виртуальных окружениях. Это означает, что когда пользователь запускает лаунчер без указания конкретного интерпретатора, он будет автоматически использовать текущее виртуальное окружение, если оно активно. Это улучшение позволяет избежать путаницы между командами, используемыми в обычной среде Python и в виртуальных окружениях, где требовалось бы использовать разные команды для одной и той же операции. В документе подробно рассматриваются сложные моменты, связанные с использованием виртуальных окружений на Windows, и предлагается решение через использование переменной окружения VIRTUAL_ENV. ПEP 486 также описывает, как это изменение повлияет на запуск скриптов и взаимодействие с командной строкой, обеспечивая более удобный и последовательный опыт для пользователей Python.
