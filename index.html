Practical use of the null garbage collector

В данной статье рассказывается о забавном практическом применении так называемого "нулевого сборщика мусора". Автор делится историей, в которой разработчик столкнулся с проблемами утечек памяти в программном обеспечении для ракеты. Интересно, что вместо устранения утечек, команда разработчиков осознавала, что их программа будет работать достаточно короткое время, и поэтому они с запасом рассчитали, сколько памяти оно может "протерять". К этому количеству они добавили дополнительную память, чтобы обеспечить работоспособность до конца заданной миссии. Эта история подчеркивает креативный подход к решению вопросов управления памятью и позволяет взглянуть на сложные аспекты разработки программного обеспечения с юмором и легкостью.

How does life happen when there's barely any light?

В этой статье исследуются удивительные возможности фотосинтеза в условиях крайнего недостатка света, таких как полярная ночь в Арктике. Учёные под руководством биогеохимика Клары Хоппе провели исследования, находясь на борту ледокола в течение нескольких месяцев, чтобы выяснить, как микроводоросли способны выживать и даже размножаться при освещении, которое в сотни раз меньше солнечного. Результаты их работы показывают, что природа может быть более продуктивной в темноте, чем считалось ранее. Научные наблюдения подтвердили теоретические модели, согласно которым фотосинтез может происходить при уровне света, близком к минимуму, необходимому для жизнедеятельности клеток. Это открытие может значительно изменить представление о жизненных циклах арктических организмов и их взаимодействиях внутри экосистемы, открывая новые горизонты для понимания микроорганизмов, существующих в условиях, казалось бы, непригодных для жизни.

There's Math.random(), and then there's Math.random() (2015)

В статье обсуждается функция Math.random() в JavaScript, которая используется для генерации случайных чисел. Основное внимание уделяется реализации этого метода в движке V8 и особенностям его работы, основанным на псевдослучайных генераторах чисел (PRNG). Объясняется, что из-за ограниченной внутренней структуры, используемой в предыдущей реализации, числа могли повторяться, что плохо сказывалось на качестве генерации. Ключевым моментом является переход на более современный и качественный алгоритм xorshift128+, который обеспечивает гораздо более длительный период генерации и высокое качество случайных значений. Однако подчеркивается, что даже с улучшенной реализацией функция не является криптографически безопасной, что ограничивает ее применение в задачах, связанных с безопасностью, таких как шифрование. Вместо этого рекомендуется использовать Web Cryptography API для следующих операций, требующих большей степени защиты.

Цель этого эксперимента — проверить, могут ли такие ответы улучшить качество взаимодействия пользователей с платформой, помочь сократить количество неотвеченных вопросов и повысить уровень участия в сообществе. Кроме того, поднимаются вопросы о моральной ответственности перед сообществом, лицензировании созданного контента и недостаточной прозрачности в процессе эксперимента. Большое внимание уделено объяснению низкоуровневых операций и тому, как базовые команды соотносятся с функциональностью процессора. Уникальные примеры позволяют освоить основные концепции ассемблера, начиная с простейших программ и заканчивая более сложными проектами. Эта книга – отличный старт для тех, кто хочет углубиться в программирование на ассемблере и понять работу компьютера на более глубоком уровне.

OpenLDK: A Java JIT compiler and runtime in Common Lisp

OpenLDK — это компилятор Just-In-Time (JIT) и среда выполнения для Java, полностью реализованные на языке Common Lisp. Хотя OpenLDK не предназначен для высокопроизводительного выполнения Java-программ, он предлагает полезный инструмент для использования отдельных Java-библиотек в Lisp-ориентированных проектах, минимизируя при этом необходимость в внешней Java-среде выполнения. С поддержкой интеграции классов Java в систему объектов Common Lisp (CLOS), проект предоставляет ряд возможностей для разработчиков, стремящихся к упрощению своей работы с библиотеками Java в Lisp.
