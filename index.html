Apache Iceberg

Apache Iceberg™ представляет собой высокопроизводительный формат для работы с крупными аналитическими таблицами. Система обеспечивает надежность и простоту SQL-таблиц в контексте больших данных, позволяя таким движкам, как Spark, Trino, Flink и другим, одновременно и безопасно работать с одними и теми же таблицами. Iceberg поддерживает гибкие SQL-команды для слияния новых данных, обновления существующих записей и выполнения целевых удалений. Ключевыми особенностями Iceberg являются возможность эволюции схемы без необходимости переписывания таблиц, скрытая партиционирование для оптимизации запросов и функция временного перемещения для воспроизводимости запросов. Эти характеристики делают Iceberg идеальным решением для управления и анализа больших данных.

An invalid 68030 instruction accidentally allowed the Mac Classic II to boot

Статья посвящена неожиданному открытию, которое произошло во время исследования процесса загрузки Macintosh Classic II. Автор, погруженный в детали работы старинной технологии, обнаружил, что ошибка в коде ROM привела к тому, что одна невалидная инструкция процессора MC68030 вместо того, чтобы вызывать сбой, фактически позволила компьютеру успешно загружаться. Это возникло благодаря тому, что инструкция неожиданно исправляла значение регистра A1, которое было неправильно установлено на момент загрузки, предотвращая таким образом сбой системы.
